# AWS Cloud Technical Essentials - Learning Repository

## 📌 Project Overview
This repository serves as a comprehensive technical documentation and portfolio of my learning journey through the **AWS Cloud Technical Essentials** course. The objective of this repository is to demonstrate a solid understanding of fundamental Amazon Web Services (AWS) concepts, ranging from compute and storage to networking and security.

As a professional focused on scalable infrastructure, I have documented the core architectural patterns and implementation details of key AWS services used to build highly available and cost-effective cloud solutions.

## 🛠 Tech Stack & Services Covered
- **Compute:** Amazon EC2, AWS Lambda, Amazon ECS, Amazon EKS
- **Storage:** Amazon S3, Amazon EBS, Amazon EFS
- **Networking:** Amazon VPC, Subnets, Security Groups, Route 53
- **Databases:** Amazon RDS (Relational), Amazon DynamoDB (NoSQL)
- **Monitoring & Governance:** Amazon CloudWatch, AWS IAM (Identity and Access Management)
- **Scaling:** Elastic Load Balancing (ELB), Auto Scaling

## 🚀 Key Learning Objectives
- **Infrastructure as a Service (IaaS):** Understanding the shared responsibility model and global infrastructure (Regions & Availability Zones).
- **Compute Optimization:** Selecting the right compute resources based on workload requirements (Serverless vs. Instance-based).
- **Network Topology:** Designing secure VPC architectures with public and private subnets.
- **Data Persistence:** Implementing appropriate storage classes and database engines for performance and durability.
- **Security Best Practices:** Applying the Principle of Least Privilege (PoLP) using IAM policies and roles.
